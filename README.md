# 📊 Sales Data Analysis using Python
## 🔷 Project Overview
This project focuses on analyzing a retail sales dataset to identify key business insights such as sales trends, regional performance, and product category analysis. The analysis is performed using Python and popular data analysis libraries.

## 🔷 Objectives
* Analyze overall sales and profit performance
* Identify top-performing regions and categories
* Understand customer segment contribution
* Discover trends and patterns in sales data
* Provide actionable business insights

## 🔷 Technologies Used
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## 🔷 Dataset
* Dataset used: *Superstore Sales Dataset*
* Contains information such as:
  * Order Date
  * Sales
  * Profit
  * Region
  * Category
  * Customer Segment

## 🔷 Steps Performed
### 1️⃣ Data Loading
* Loaded dataset using Pandas
* Previewed data using `.head()`

### 2️⃣ Data Cleaning
* Handled missing values
* Removed duplicate records
* Converted date columns to datetime format

### 3️⃣ Feature Engineering
* Created new columns:
  * Year
  * Month
  * Month Name

### 4️⃣ Exploratory Data Analysis (EDA)
* Analyzed:
  * Sales by Region
  * Sales by Category
  * Profit by Region
  * Sales by Customer Segment
* Used `groupby()` operations for aggregation

### 5️⃣ Data Visualization
* Bar charts for category and regional sales
* Line charts for monthly trends
* Scatter plots for Sales vs Profit
* Heatmap for correlation analysis

## 🔷 Key Insights
* West region generated the highest sales
* Technology category contributed the most revenue
* High discounts negatively impacted profit
* Certain products dominated overall sales
* Sales showed seasonal trends across months

## 🔷 Conclusion
This project demonstrates how data analysis can help businesses make better decisions by identifying trends, improving profitability, and focusing on high-performing areas.
